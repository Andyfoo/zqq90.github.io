Webit Script 读音

Webit Script /wɛbɪt skrɪpt/, 或者拆成 we bit script

这是一个开源的Java模板引擎

Webit Script 是一个简单灵活自由开放的开源的模板引擎，基于Java开发，支持Java5 及以上版本，采用BSD开源协议。

JavaScript风格的语法

Webit Script 尽量更多的采用javascript的 标准语法，较少前端开发者的学习难度，同时你可以使用javascript的高亮方案高亮显示您的模板。
